🚀 Generative AI Suite: Chatbots, RAG, LangChain Agents & Voice
A multi-modal Generative AI chatbot that supports general conversation, PDF-based Q&A, web scraping, and voice responses, built with a modern UI using Gradio.

📌 Project Description
The Generative AI Suite is an intelligent conversational platform designed to deliver fast, accurate, and context-aware responses.
Users can interact with the chatbot through normal text queries, upload PDF documents for question answering, or provide website URLs for live information extraction.
The system also supports text-to-speech (voice output) with a toggle option for enhanced accessibility.

✨ Key Features
💬 General Chat Responses – Natural and fast conversational AI
📄 PDF Reading & Question Answering – Extracts and answers questions from uploaded PDFs
🌐 Web Scraping Integration – Fetches and summarizes content from live websites
🔊 Voice Output (Text-to-Speech) – AI responses available in both text and audio formats
⚡ Fast Response Time – Optimized for quick replies across all modes
🎨 Premium UI/UX – Clean, modern, and user-friendly interface
🛠️ Technologies Used
Python
Gradio – User Interface
LangChain
Google Gemini API
BeautifulSoup – Web scraping
PyMuPDF (fitz) – PDF processing
gTTS – Text-to-Speech
HTML & CSS – UI Styling
